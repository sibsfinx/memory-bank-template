# Flows And Templates Index

The `memory-bank/flows/` directory contains the reusable process layer for the template: lifecycle rules, stable identifier taxonomy, and governed templates.

- [Task Workflows](workflows.md) — task routing by type, base development cycle, and autonomy gradient.
- [Epic Flow](epic-flow.md) — lifecycle of large initiatives: roadmap, decision log, risks, and handoff into feature packages.
- [Feature Flow](feature-flow.md) — lifecycle `brief.md -> optional design.md -> implementation-plan.md`, gates, and stable IDs (`REQ-*`, `SOL-*`, `STEP-*`).
- [Templates Index](templates/README.md) — reference templates for governed documents including PRD, use case, epic, feature, and ADR.
